# Estrutura de Imagens

Coloque suas imagens nesta pasta seguindo esta estrutura:

## Hero Section
- `hero-background.jpg` (1920x1080) - Imagem de fundo do Hero

## Membros/Avatares
- `member1.jpg` (300x300) - Avatar membro 1
- `member2.jpg` (300x300) - Avatar membro 2
- `member3.jpg` (300x300) - Avatar membro 3
- `member4.jpg` (300x300) - Avatar membro 4

## Parceiros/Logos
- `partner1.png` - Logo parceiro 1
- `partner2.png` - Logo parceiro 2
- `partner3.png` - Logo parceiro 3
- `partner4.png` - Logo parceiro 4
- `partner5.png` - Logo parceiro 5

## About Section
- `about.jpg` (800x600) - Imagem principal da seção Sobre

## Bento Grid
- `bento-large.jpg` (800x800) - Card grande
- `bento-small1.jpg` (400x400) - Card pequeno 1
- `bento-small2.jpg` (400x400) - Card pequeno 2
- `bento-small3.jpg` (400x400) - Card pequeno 3

## Categorias
- `category1.jpg` (600x400) - Categoria 1
- `category2.jpg` (600x400) - Categoria 2
- `category3.jpg` (600x400) - Categoria 3
- `category4.jpg` (600x400) - Categoria 4

## Depoimentos
- `testimonial1.jpg` (300x300) - Avatar depoimento 1
- `testimonial2.jpg` (300x300) - Avatar depoimento 2
- `testimonial3.jpg` (300x300) - Avatar depoimento 3

## SEO/Social
- `preview.jpg` (1200x630) - OG Image para redes sociais
- `logo.png` (opcional) - Logo do site

## Outros
- `favicon.ico` - Ícone do navegador

---

**Dica:** Use imagens otimizadas em formato WebP ou JPG para melhor performance.
